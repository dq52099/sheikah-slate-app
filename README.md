# Sheikah Slate: Image Link (希卡之石：图影链接)

这是一个基于 `boxying-image-gateway` 的图像创作 App，其交互逻辑与视觉完全适配《塞尔达传说：旷野之息》中的“希卡之石”终端。

## 🏺 视觉方案 (Sheikah Identity)
- **配色 (Palette)**:
  - `Sheikah Blue`: #00D2FF (希卡能量流动的蓝色)
  - `Ancient Core`: #FF9600 (古代核心的橙金光芒)
  - `Slate Stone`: #1A1A1A (底座石材的深灰色)
  - `Guardian Green`: #00FF99 (生命力与森林的绿色)
- **UI 元素**:
  - **点阵引导线**: 模拟希卡感应器的动态线条。
  - **断裂边框**: 具有年代感的石碑破碎效果。

## 📜 核心术语 (Glossary)
| 原术语 | BotW 术语 | 描述 |
| :--- | :--- | :--- |
| Generate | **具现化 (Materialize)** | 消耗能量电池，将古老符文转化为图影。 |
| Edit | **时停回溯 (Recall)** | 选定图影，回溯至创作瞬间并进行修改。 |
| Prompt | **远古符文 (Runes)** | 用于指引能量具现的具体指令。 |
| Quota | **能量电池 (Energy Cells)** | 希卡之石的能源储备。 |
| History | **希卡图鉴 (Compendium)** | 记录海拉鲁大陆（或你的创作）的一切影像。 |

## 🕹 交互体验
- **扫描感应**: 在生图时，界面会伴随希卡感应器的圆环扫描动效。
- **解锁仪式**: 首次进入时，会有类似于“地图下载完成”的认证动画。
- **神庙入口**: 所有的设置项都被设计为神庙内部的机关样式。
